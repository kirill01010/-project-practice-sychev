# Описание BPMN
# BPMN процесс

## Процесс: оформление заказа (Яндекс Маркет)

## Участники:
- Покупатель
- Яндекс Маркет
- Продавец
- Курьер

## Описание:
Покупатель выбирает товар, система проверяет наличие, продавец подтверждает заказ, курьер доставляет товар.

## Подпроцессы:
- Проверка наличия товара
- Доставка товара

## Результат:
- заказ выполнен или отменён

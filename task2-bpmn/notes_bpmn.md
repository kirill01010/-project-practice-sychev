# Заметки по BPMN

## Назначение
BPMN используется для описания бизнес-процессов и их участников.

## Основные элементы
Start Event — начало процесса  
End Event — завершение  
Task — действие  
Gateway — условие  
Subprocess — подпроцесс  
Sequence Flow — поток действий  
Message Flow — обмен между участниками  

## Шлюзы
XOR — один вариант  
AND — параллельные процессы  
OR — один или несколько вариантов  

## Правила
- один Start Event
- корректные подписи условий
- Message Flow только между пулами
